--- crudBACK-master {
Para que ande le hice:
npm i @angular/cdk --save

Modifiqué el tsconfig.app.json del crudFRONT-master, le agregué la sentencia "angularCompilerOptions": { "enableIvy": "ngtsc" }
el tsconfig.app.json quedó asi:
{
  "extends": "./tsconfig.json",
  "compilerOptions": {
    "outDir": "./out-tsc/app",
    "types": []
  },
  "angularCompilerOptions": { "enableIvy": "ngtsc" },
  "include": [
    "src/**/*.ts"
  ],
  "exclude": [
    "src/test.ts",
    "src/**/*.spec.ts"
  ]
}
}
